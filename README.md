# smart-restaurant
基于Java+Springboot+Vue的智能餐厅服务系统
一、系统介绍
本项目前后端分离，分为管理员、用户两种角色
1、用户：
- 注册、登录、智能点餐、订单查询、个人信息
2、管理员：
- 菜单管理、用户管理、角色管理、分类管理、菜品管理、订单管理、标签管理、座位管理、客服管理、订单管理
产品概念：提升餐厅运营效率和顾客体验
核心功能详述：
1、智能点餐
扫码点餐和语音点餐双模式
菜品智能推荐
特殊需求识别
2、后厨优化
订单智能排序
出餐时间预测
库存预警管理
3、服务提升
顾客情绪识别
服务质量评估
反馈自动分析
4、技术实现路径：
开发微信小程序点餐系统
使用推荐算法实现菜品推荐
集成情感分析API（使用百度AI）
使用Redis进行缓存优化
5、交付物要求：
完整的点餐系统原型
推荐算法实现代码
系统性能测试报告
用户体验分析报告

二、所用技术

后端技术栈：

- Springboot
- MybatisPlus
- Mysql
- Maven
- redis
- springSecurity
- Jwt

前端技术栈：

- Vue 
- Vue-router 
- axios 
- element-ui

基础环境 :IDEA, JDK17, Mysql8.0,Maven3.9, node10.9，navicat
